# Auth Module

API using Jest(NodeJS) to Authenticate User.

This module uses the technologies below:

- Docker
- MongoDB
- NodeJS

## Requirements

- Docker v19.03.6
- DockerCompose v1.24.1

## How to run

- Lift stack

```sh
docker-compose up
```

- Lift stack using decoupled mode

```sh
docker-compose up -d
```

## Provides

- [API](./api)
- Database
