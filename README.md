## Aplicação de Contas

Aplicação rest de contas desenvolvido em Golang.

**Para rodar**

Rode `docker-compose up --build -d` dentro da pasta do projeto

A aplicação está disponível também em ambiente heroku https://contas-food.herokuapp.com/
